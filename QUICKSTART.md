# 卓球成績管理システム - クイックスタートガイド

## 🚀 5分でできる初期設定

### ステップ1: プラグインのインストール

1. WordPressの管理画面にログイン
2. 「プラグイン」→「新規追加」→「プラグインのアップロード」
3. `tabletennis-stats-plugin.zip` をアップロード
4. 「今すぐインストール」をクリック
5. インストール完了後、「プラグインを有効化」をクリック

### ステップ2: 選手を登録する

1. 管理画面の左メニューに「卓球成績」が表示されます
2. 「卓球成績」→「選手管理」→「新規追加」
3. 必要な情報を入力:
   - **選手名**（必須）
   - ふりがな
   - **性別**（必須）
   - 出身（都道府県）
   - 戦術（右ペン、左ペンなど）
   - 戦術詳細
   - 参考動画URL（YouTubeなど）
4. 「登録」ボタンをクリック

**ポイント:** 参考動画は「+ 動画を追加」ボタンで複数追加できます！

### ステップ3: 試合を登録する

1. 「卓球成績」→「試合管理」→「新規追加」
2. 試合情報を入力:
   - **試合名**（必須）例: 2025年全日本選手権
   - **開催日**（必須）
   - 会場
   - **試合種別**（トーナメント/リーグ戦など）
   - 説明
3. 「登録」ボタンをクリック
4. 保存後、「参加者管理に進む」をクリック

### ステップ4: 参加者と順位を登録する

1. 試合の「参加者管理」画面で:
2. 選手を選択し、順位を入力:
   - 1 = 優勝
   - 2 = 準優勝
   - 3-4 = ベスト4
   - 5-8 = ベスト8
   - 9-16 = ベスト16
   - 99 = 予選敗退
3. 「+ 参加者を追加」で行を増やせます
4. 「保存」ボタンをクリック

### ステップ5: 対戦結果を登録する（オプション）

詳細な対戦結果も記録できます:

1. 「卓球成績」→「対戦結果」→「新規追加」
2. 情報を入力:
   - 試合を選択
   - 回戦情報（例: 決勝、準決勝）
   - 選手1と選手2を選択
   - 各選手のゲーム数（例: 4-2）
3. 「登録」ボタンをクリック

### ステップ6: 公開ページを作る

1. 「固定ページ」→「新規追加」
2. ページタイトル: 「試合結果」など
3. ブロックエディタで「ショートコード」ブロックを追加
4. 以下を入力:

**トップページ（最新試合一覧）**
```
[tt_stats_home]
```

**検索ページ**
```
[tt_stats_search]
```

5. 「公開」ボタンをクリック
6. 公開ページを確認！

## 📱 実際の使い方

### 管理者の日常業務

1. **試合後すぐ:**
   - 試合管理 → 参加者管理で順位を更新
   
2. **詳細を追加したい時:**
   - 対戦結果で個別の試合結果を追加

3. **新しい選手が出てきた:**
   - 選手管理で選手を追加
   - 動画URLがあれば一緒に登録

### ユーザーの使い方

1. **選手を探す:**
   - 検索ページで名前、性別、出身、戦術で検索
   
2. **選手の詳細を見る:**
   - 選手名をクリック
   - プロフィール、参考動画、試合結果が表示される

3. **対戦結果を調べる:**
   - 検索ページの「対戦検索」タブ
   - 2名の選手名を入力すると直接対決を表示

## 💡 便利なTips

### 選手の動画を効果的に使う

- YouTubeのURLをそのまま貼り付ければ自動で埋め込まれます
- 複数の動画を追加して試合ごとに整理できます
- タイトルと説明を入力すると分かりやすい

### 順位の入力を効率的に

- 優勝から順に入力していくとスムーズ
- 同じ順位の選手（ベスト4など）は同じ数字でOK
- 予選敗退は「99」と入力

### 検索を活用する

- 選手一覧や試合一覧に検索ボックスがあります
- 部分一致で検索できるので、名前の一部でOK
- 対戦結果は試合名で絞り込めます

## ❓ よくある質問

**Q: 選手の写真はどうやってアップロードするの？**
A: 現在は画像URLを入力する方式です。画像を先にメディアライブラリにアップロードして、そのURLをコピーして貼り付けてください。

**Q: 過去の試合データを一括で入力したい**
A: CSVインポート機能を次のバージョンで実装予定です。現在は1件ずつ入力してください。

**Q: スマホで見やすい？**
A: はい！スマホ表示に最適化されています。

**Q: データのバックアップは？**
A: WordPressのデータベースに保存されるので、通常のWordPressバックアップで問題ありません。

**Q: 選手や試合を削除したらどうなる？**
A: 選手を削除すると関連する動画、参加記録、対戦結果も削除されます。試合を削除すると参加記録と対戦結果も削除されます。削除前に確認ダイアログが表示されます。

## 🆘 サポート

問題が発生した場合:

1. WordPress管理画面で「卓球成績」→「ダッシュボード」を確認
2. データが正しく登録されているか確認
3. ブラウザのコンソールエラーをチェック（F12キーで開く）
